Чтобы запустить программу необходимо установить базу данных "testdb.backup", через pgAdmin:
	1. Запустите pgAdmin.
	2. Подключитесь к серверу.
	3. Щёлкните правой кнопкой на Databases → Restore.
	4. Укажите путь к вашему файлу .backup.
	5. Настройте параметры (например, Restore Owner, No ACL) и нажмите Restore.
Также необходимо проверить параметры подключения из файла db_config.json.
После этого запустить "arrears.exe".
Если программа сразу же закрывается проверьте ошибку через командную строку:
	1. Запустите командную строку.
	2. Перейдите в папку с файлом .exe через команду "cd 'путь к файлу'".
	3. Запустите "arrears.exe".
Скорее всего ошибка будет связана с неправильно прописанными настройками файла конфигурации "db_config.json". Проверьте правильно ли прописан пользователь, пароль, имя базы, порт и хост.
